VAR:var has a function level scope. this means that a variable declared with var is visible throughout the entire funciton in which it is declared. if the var declaretion is outside of any function, it becomes a global variable and is accessible throughout the entire program..
LET:the let keyword in js introduces block level scoping. which means that a variable decalared with let is only accessible with in the block (a pair of curly braces)in which it is defined. this cludes blocks inside functions,loops and conditional statements..
CONST:variable declared with const are confined to the block in which they are defined. variables declared with const must be assigned a value upon delaretion and once assigned their value cannot be changed. while the variable itself cannot be ressigned for object and arrays declared with const their properties or elements can still be modifed..
